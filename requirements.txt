# WhatsApp Android → iOS Migration Tool
# ======================================
# 
# This project uses ONLY Python Standard Library modules.
# No external dependencies required for core functionality.
#
# Python Version: 3.8+
#
# Core Standard Library Modules Used:
# - sqlite3: Database manipulation (Android msgstore.db, iOS ChatStorage.sqlite)
# - subprocess: ADB command execution
# - pathlib: Cross-platform path handling
# - logging: Structured logging to files and console
# - json: Parsing iTunes backup Manifest.db
# - shutil: File operations and backups
# - datetime: Timestamp conversions
# - zipfile: .ab backup extraction (if using Python implementation)
# - struct: Binary data parsing
# - argparse: CLI argument parsing
# - typing: Type hints for better code quality
#

# ======================================
# External Binary Dependencies
# ======================================
# These are NOT Python packages. Download manually:
#
# 1. Android Debug Bridge (ADB):
#    - Windows: https://developer.android.com/tools/releases/platform-tools
#    - Extract adb.exe, AdbWinApi.dll, AdbWinUsbApi.dll to bin/
#    - See: bin/README.md for detailed instructions
#
# 2. Legacy WhatsApp APKs:
#    - Versions: 2.11.431 - 2.11.498 (required for Android backup)
#    - Source: APKMirror or APKPure (legal download only)
#    - Place in: apk/
#    - See: apk/README.md for legal considerations
#
# 3. iTunes (Windows) or Finder (macOS):
#    - For iOS device backup creation
#    - https://www.apple.com/itunes/
#
